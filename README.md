# DEMO-CI-DC-Project

## CI/CD Pipeline Project

This project demonstrates a complete CI/CD pipeline using:

- Jenkins
- GitHub
- Docker
- Nginx

## Workflow

GitHub → Jenkins → Docker Build → Container Deployment

## Tools Used

- Jenkins
- Docker
- Git
- CentOS 9
